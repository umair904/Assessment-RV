## Instructions/Information
-> To run the code you have to clone the project to your local repository and then you have to follow the (Installation) steps given below.
-> Then you have to create a mongoDB Cluster Online, visit mongoDB website > Sign UP/ Sign IN > Create Clustor > Get connection link/URI.
-> Create a .env file at root directory (copy the content in sampleENV.txt file).
-> Then execute one command from (Running the app) section below: > Watch Mode command is recommended.
-> Then idealy the project will be working on your local machine.
-> Then use the postman collection link given to you in email for the testing the APIs.

---- Assumptions Made in Project ----
-> User have to select the Categories of his/her interests using API (Like Pinterest, Twitter).
-> Movie can only exist in one category at a time.

---- Note ----
-> Comments are added in the code where necessary. 
-> I used chat-Gpt at some points for syntax. for example [ (existingCategory as any)._id ]

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
